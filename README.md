BUDGET NEST:

Budget Nest is a full-stack budget management web application that helps users track expenses, analyze spending patterns, and manage personal finances efficiently. The application provides secure authentication and visual analytics to help users understand and improve their financial habits.

🚀 Features

User Registration & Login

JWT Authentication

Google OAuth Login

Add, Edit, Delete Expenses

Weekly & Monthly Expense Reports

Category-wise Expense Breakdown

Pie Charts & Bar Graph Analytics

Custom User Settings

🛠️ Tech Stack

Frontend:
React / Bootstrap

Backend:
Node.js
Express.js

Database:
MySQL / SQLite

Authentication:
JWT (JSON Web Token)
Google OAuth
